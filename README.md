# نظام تصنيف للتشغيل - HTML + Supabase V3

## خطوات الرفع
1. في Supabase افتح SQL Editor وشغل ملف `schema_update_v3.sql` مرة واحدة.
2. ارفع هذه الملفات إلى GitHub Pages:
   - index.html
   - admin.html
   - supervisor.html
   - app.js
   - style.css
   - README.md
3. لا ترفع ملف `schema_update_v3.sql` للموقع، هو خاص بـ Supabase فقط.
4. افتح الموقع واضغط Ctrl + F5.
