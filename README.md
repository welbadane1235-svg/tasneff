# تطبيق تصنيف HTML + Supabase

هذه نسخة جديدة من الصفر بملفات HTML عادية، مناسبة للرفع على GitHub Pages.

## الملفات

- `index.html` تسجيل الدخول
- `admin.html` نسخة الإدارة
- `supervisor.html` نسخة المشرف
- `app.js` الربط والمنطق
- `style.css` التصميم
- `schema.sql` جداول Supabase

## طريقة التشغيل

1. افتح Supabase.
2. ادخل إلى SQL Editor.
3. انسخ محتوى `schema.sql` واضغط Run.
4. افتح ملف `app.js` وعدّل:

```js
const SUPABASE_URL = "رابط مشروعك";
const SUPABASE_ANON_KEY = "المفتاح العام";
```

5. ارفع الملفات كلها على GitHub.
6. فعل GitHub Pages من إعدادات المستودع.

هذه نسخة تشغيلية أولى ثابتة، وليست النسخة الأمنية النهائية. لأنها HTML فقط، تم فتح سياسات Supabase للتجربة السريعة. بعد اعتماد الشكل النهائي، الأفضل تحويل تسجيل الدخول إلى Supabase Auth مع صلاحيات RLS مغلقة.
